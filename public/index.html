Email signature

Fill in your details, then copy — Gmail needs rich paste (not raw code).

Where to paste: Gmail → Settings → Signature. Outlook on the web → Mail → Signatures (not the new-message body). Outlook often needs the copy button (it grabs the rendered preview first). If you still see code: click inside the preview, ⌘A / Ctrl+A, copy, paste into the signature box.

Preview

Gmail: Gear → See all settings → General → scroll to Signature → create/edit → paste there. Then new emails use that signature automatically. Do not paste into the compose window to install it. If paste shows code in Signature settings: use Chrome, use our copy button, or copy from the preview (⌘A / Ctrl+A) and paste again. Outlook web: Settings → Mail → Signatures → edit — paste there. If the editor shows raw tags, use the copy button (preview-based) or copy from the preview area after selecting all.
